# GestionSolicitudes
Sistema de gestion de solicitudes usando enumeradores y listas en C#
# Explicación del uso del Enum
Utilicé el enumerador `EstadoSolicitud` para restringir las opciones de estado a: Pendiente, En Proceso, Completada y Cancelada. Esto elimina el uso de "textos libres" y asegura que la aplicación solo maneje valores válidos mediante el proceso de casting: `(EstadoSolicitud)valor_entero`.
